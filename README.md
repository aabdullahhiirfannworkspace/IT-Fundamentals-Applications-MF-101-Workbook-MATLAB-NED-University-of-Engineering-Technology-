# IT Fundamentals Applications MF-101 Workbook MATLAB Solution NED University of Engineering & Technology

## Overview

This repository contains MATLAB solutions for the **IT Fundamentals Applications (MF-101) Workbook** used at **NED University of Engineering & Technology** in the **2026 Spring Semester**. It is intended as a study resource to help students understand MATLAB programming concepts, practice problem-solving techniques, and compare their work with completed solutions.

## Contents

* MATLAB script files (`.m`) for workbook exercises
* Practice solutions for selected MF-101 problems
* Examples covering MATLAB fundamentals, matrices, arrays, functions, plotting, and basic programming
* Reference material for learning MATLAB

## Purpose

The materials are provided for educational purposes only. Students are encouraged to attempt each exercise independently before referring to the solutions.

## Requirements

* MATLAB (R2022a or later recommended) or GNU Octave
* Some files may not display correctly on GitHub. It is recommended to download the files and open them in MATLAB or a compatible editor.

## Disclaimer

**Errors and omissions excepted.** These solutions are shared solely to support learning and should not be submitted as original coursework. Please follow your instructor's academic integrity guidelines and the policies of NED University of Engineering & Technology.
